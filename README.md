# Network Packet Analyzer

A simple Python-based Network Packet Analyzer developed as part of the Prodigy InfoTech Cyber Security Internship.

## Features
- Analyzes packet information
- Displays Source and Destination IP Addresses
- Identifies Protocol Type
- Examines Payload Data
- Calculates Payload Length

## Technologies Used
- Python 3

## Purpose
This project helps in understanding basic network packet analysis concepts and cybersecurity fundamentals.
